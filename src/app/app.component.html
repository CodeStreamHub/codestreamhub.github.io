How to install
Download the CLI (https://github.com/CodeStreamHub/codestream-cli/tree/main/binaries) and add to PATH
Run 'cs login -t <API token>' command to login to your org/instance (This is a one-time step)
Run 'cs ci list' to know the available custom integrations and pick the id of required integration
Run 'cs ci install <integration id> -p <project>'
Selected integration is now installed to your org/instance and you can start using it.
And for upgrading the existing integrations, just use 'cs ci upgrade' command with appropriate values.
For any help related to CLI, use -h flag.

How to Install/Upgrade
Download the CLI and add to PATH
Run 'cs login -t <API token>' command to login to your org/instance (This is a one-time step)
Run 'cs ci install {{integrationId}} -p <project>'

Selected integration is now installed to your org/instance and you can start using it. And for upgrading the existing integrations, just use 'cs ci upgrade' command with appropriate values.
For any help related to CLI, use -h flag.
*Copy/Click ID from the card to replace <integration id> value
